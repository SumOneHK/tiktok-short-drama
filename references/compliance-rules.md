## 说明

本文件用于创作阶段的前置筛查，不替代法务、发行审核或平台最终判定。出现高风险时，优先改内容，不赌审核。

## 通用红线

> 以下条目为**逐项打钩**结构，`/开始` 合规前置快筛、`/合规` 终筛必须按 ID 逐项核对；命中任意 R 类红线 = P0，禁止推进。

| ID | 红线条目 | 判定要点 | 判级 | 安全改写出口 |
|----|---------|---------|------|-------------|
| R-01 | 未成年人性相关内容 | 任何角色未满 18 岁的性暗示、性场景、性化描写 | P0 | 改为成年角色或彻底移除性化元素 |
| R-02 | 强奸、美化胁迫性关系、乱伦 | "霸道占有""强行亲密"作为浪漫桥段；血亲恋爱 | P0 | 改为双向选择 + 明确同意；血亲改为非血亲 |
| R-03 | 明确仇恨、歧视、种族/宗教攻击 | 把种族、国籍、宗教、性取向、残障作为侮辱笑点或反派标签 | P0 | 冲突针对个体行为而非群体身份 |

**逐项核对要求**：

- `/开始` 合规前置快筛：候选方向必须按 R-01 ~ R-06 逐项给出"通过 / 命中"标记，命中任意一项直接踢出
- `/合规` 终筛：剧本中每场戏必须可溯源到通过的红线 ID，命中即生成 P0 修改单

## `mode=overseas` 审核重点

针对 TikTok/海外平台，重点筛查以下类别：

1. 暴力与血腥
   可以有冲突，但避免细节化、感官化、猎奇化呈现。
2. 性与裸露
   不写未成年相关，不写胁迫，不依赖露骨描写做卖点。
3. 仇恨与骚扰
   不把种族、国籍、宗教、性取向、残障等作为侮辱笑点。
4. 危险行为
   不把可模仿的危险行为写成值得效仿的爽点。
5. 自杀自残与精神危机
   不把自伤作为勒索爱情或卡点工具。
6. 非法活动与诈骗
   不把犯罪细节写成教程。
7. 误导与仿冒
   不借现实公众人物、真实机构做伪造情节营销。
8. 未成年人保护
   涉及未成年人时，避免性化、虐待细节化、危险模仿场景。

海外模式额外要求：

- 暴力、性、危险行为不能成为主要付费卡点
- 不用高度敏感现实政治议题做流量钩子
- 不用羞辱弱势群体换取喜剧效果
- 不把胁迫式亲密、跟踪控制、明显虐待包装成浪漫高潮

## `mode=domestic` 审核重点

国内模式除通用红线外，重点筛查：

1. 导向风险
   不宣扬极端拜金、权钱崇拜、恶性复仇、违法致富。
2. 低俗与擦边
   不以露骨暧昧、性羞辱、婚恋伦理奇观做核心卖点。
3. 违法与失德正当化
   不把校园霸凌、家暴、PUA、骗婚等包装成浪漫。
4. 封建迷信和伪科学误导
   可有虚构奇幻设定，但不能鼓励现实迷信崇拜、非法改运、邪教式叙事。
5. 价值导向失衡
   不能只写报复快感，不写后果与边界。
6. 标题与宣传失范
   不使用明显低俗、夸张煽动、羞辱性标题。

## 审核流程

### 1. 立项预筛

检查 premise、题材、主关系是否天然高风险；若风险来自题材底座，应尽早换题。

### 2. 结构复筛

检查高风险内容是否被拿来做爽点、反转或卡点；尤其检查：

- 强迫亲密
- 自伤威胁
- 极端暴力复仇
- 弱势群体羞辱
- 未成年人危险模仿场景

### 3. 剧本终筛

逐集检查：

- 对白是否越线
- 场面是否细节化越线
- 结尾钩子是否使用高风险刺激
- 海外版英文表达是否放大了原本含混的风险

## 风险分级

- `P0`：明显红线，必须删除或重写
- `P1`：有较大审核风险，需要降级表达或改情节
- `P2`：措辞、画面、卡点方式可优化

## 合规改写优先级

1. 先改 premise 和关键桥段
2. 再改卡点与高风险爽点
3. 最后改对白措辞和镜头表述

## 常用安全改写法

当内容接近红线但戏剧功能仍然需要时，优先做以下改写：

1. 胁迫式亲密 -> 改成明确双向选择，或在关键处增加拒绝边界
2. 猎奇暴力 -> 改成结果导向的后果展示，减少细节化过程描写
3. 自伤威胁卡点 -> 改成离开、揭发、公开羞辱、利益断裂等非自残型代价
4. 仇恨或歧视笑点 -> 改成针对行为、权力或个人关系的冲突，不打敏感群体标签
5. 犯罪教程化桥段 -> 改成模糊化执行细节，只保留结果和后果
