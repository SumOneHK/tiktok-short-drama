# 语言模式规则

## 总原则

中文团队使用的策划和说明文档优先保持中文，避免因为全英文导致沟通成本上升。

## `mode=overseas`

用于面向 TikTok/海外平台的剧本版本。

写作规则：

- `选题分析.md`、`故事设定.md`、`故事结构.md`、`分集大纲.md`、所有质检与合规文档：中文
- `分集剧本/*.md` 中的标题、场景、动作、调度、情绪说明：中文
- `分集剧本/*.md` 中的对白、旁白、`OS`、`VO`、屏幕字卡：可用英文，但一旦使用英文，必须紧跟中文翻译

英文对白要求：

1. 口语化，可直接表演。
2. 句子短，少从句，少翻译腔。
3. 避免把中文成语、网络梗直译成英文。
4. 情绪词优先直接，不绕弯。
5. 同一角色保持稳定说话方式。

英文翻译要求：

1. 英文原文后必须紧跟中文翻译，不得省略。
2. 中文翻译以准确传达情绪、关系和潜台词为先，不要求逐词直译。
3. 对白、旁白、`OS`、`VO`、屏幕字卡都适用同一规则。
4. 推荐写法为英文原文一行，下一行写 `中译：...`；若模板空间有限，也可同一条目内写成 `英文（中译：中文）`。
5. 若英文里出现双关、俚语或不可直译表达，中文翻译要优先还原戏剧功能，而不是硬译字面。

英文对白执行约束：

- 单句默认优先控制在 4-14 个英文词；超过 18 个词时，要怀疑是否写成了解释性台词
- 能用动作、停顿、打断表达的信息，不要全部塞进对白
- 优先使用可表演、可配音的自然表达，例如收缩形式、短句、反问、打断句
- 避免中文式强调，如 `very very`、`you dare`、`how could you possibly`
- 同一角色在同一场里不要同时出现书面腔和网络俚语腔

命名建议：

- 中文内部名 + 英文展示名可并存
- 第一次出场可写：`林晚（Lena Lin）`
- 若题材强海外化，可直接用英文名，但设定说明仍用中文

## `mode=domestic`

全部文档、全部剧本内容均使用中文。

## 中文对白质量标准

无论国内还是出海模式，策划文档和说明部分的中文对白（或中文版剧本对白）必须满足以下标准：

### 长度控制
- 单句对白默认不超过 25 个汉字；超过 30 字时，要怀疑是否写成了解释性台词
- 一口气超过 3 句没有被打断的独白，视为节奏风险——考虑用动作、反应镜头打断
- 强调可用重复（"不行，绝对不行"），不用叠加修饰词（"非常非常地愤怒"）

### 角色声线区分
每个角色的说话方式必须可区分，至少在以下维度上体现差异：

| 声线类型 | 常见特征 | 禁止混用 |
|---------|---------|---------|
| 强势/霸道型 | 短句、命令式、少解释 | 不突然变成解释型工具人 |
| 草根/平民型 | 口语化、有方言色彩词汇、情绪外露 | 不突然说出精英式术语 |
| 腹黑/高智型 | 反问多、留余地、信息密度高 | 不为推剧情而犯低级口误 |
| 恶毒型配角 | 讽刺性、强调对比贬低、喜欢旁观者 | 不突然变善良 |
| 暗恋/压抑型 | 欲言又止、转移话题、否认式表达 | 不在揭示前就直白说出来 |

### 信息堆砌禁止
- 一句台词不能同时传递超过 2 个关键信息
- 不允许用台词替代应有的动作表达（如："她站起来，走向门口，说：'我要走了。'" → 直接写动作行）
- 背景设定信息不通过角色之间的"解释性对话"传递（你也知道，我们家三年前……）

### 情绪词直接原则
- 优先使用可配音可表演的直接情绪词，不用迂回替代
- 例：用"你敢！"而不是"你怎么可以这样对我"（除非是刻意压抑的角色）

## 剧本中的语言自检

1. 海外模式下，是否只有对白、旁白、`OS`、`VO`、屏幕字卡使用英文？其余说明是否保持中文？
2. 海外模式下，凡英文对白、旁白、`OS`、`VO`、屏幕字卡，是否都紧跟了中文翻译？
3. 海外模式下，`OS`、`VO` 是否也保持英文原文 + 中文翻译？
4. 英文对白是否自然，不像中文逐字翻译？
5. 情绪表达是否清晰到可以直接配音或表演？
6. 是否混入对海外观众不友好的本地化梗？
7. 单句是否过长，是否存在解释性独白替代动作的问题？
