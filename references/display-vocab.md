# 用户展示口径

> **何时读本文件**：
> - 任何要把内部状态枚举值翻译成中文给用户看的时候
> - `/开始` `/立项` 阶段对外输出推荐方案、配置建议时
> - 写 `选题分析.md` 推荐版/正式版需要展示字段时
>
> 仅在写 `剧本状态.json` 内部字段、或在文档里说明"模板切换条件"时才使用内部枚举值；面向用户的所有输出一律走中文翻译。

## 翻译表

对用户展示推荐方案时，一律优先使用中文表达，不直接抛出内部枚举值。

- `overseas` 对用户写为：出海版
- `domestic` 对用户写为：国内版
- 本 skill 的最终交付一律是 60-90 秒竖屏 AI 视频；不把"真人/漫剧/混合"当作不同创作管线
- `artStyle` 字段对用户展示为"艺术风格"，可选值是开放列表（详见 `references/ai-production.md` 全局生成规则），常见值包括：真人感 AI 短剧 / 写实电影感 / 半写实 CG / 国漫风 / 韩漫风 / 动态漫画 / 港风复古 / 日系清冷 / 欧美大片 / 胶片颗粒 / 国潮水墨 / 赛博朋克 / 厚涂插画 / 3D 卡通
- `north-america` 对用户写为：北美市场
- `europe` 对用户写为：欧洲市场
- `sea` 对用户写为：东南亚市场
- `latam` 对用户写为：拉美市场
- `middle-east` 对用户写为：中东市场
- `mixed` 对用户写为：混合市场
- `overseas-en-dialogue` 对用户写为：中文策划，英文对白版
- `domestic-zh` 对用户写为：全中文版本

## 推荐写法示例

- 建议版本：出海版
- 建议艺术风格：真人感 AI 短剧 / 国漫风 / 动态漫画等（在 `/设定` 阶段写入 `故事设定.md` 预锁，首次 `/分镜脚本` 创建 `AI资产圣经.md` 时继承为全局生成规则）
- 建议市场：北美市场
- 建议规模：60 集，每集 90 秒
- 语言建议：中文策划，英文对白版

只有在写 `剧本状态.json` 和说明模板切换条件时，才使用内部枚举值。
