# 分镜脚本模板

## 目录

- 使用规则
- 统一硬字段
- 镜头分级与继承规则
- 时长硬约束
- AI 视频分镜模板
- 交付检查

## 使用规则

- `/分镜脚本 {起止集}` 只使用本文件模板
- 分镜脚本默认产物目录为 `分镜脚本/`
- 文件命名统一为 `第NNN集.md`
- 分镜脚本是 AI 视频生成制作稿，不替代 `分集剧本/*.md`
- 本 skill 的最终交付一律是 60-90 秒竖屏 AI 视频，所有分镜稿都用同一套镜头语言（景别 / 机位 / 运镜 / 动作起止 / 时长）拆解
- 艺术风格（写实电影感 / 港风复古 / 国潮水墨 / 赛博朋克 / 厚涂插画 / 3D 卡通 等）由 `AI资产圣经.md` 的 `artStyle` 全局生成规则统一定义，不在本文件硬编码；artStyle 切换不影响分镜字段结构，只影响 `AI画面提示` 的关键词
- 海外模式下，只有对白、旁白、`OS`、`VO`、屏幕字卡允许英文，且必须紧跟中文翻译
- 海外模式下，文档里的 `中译` 仅供团队审校；凡 `字卡/字幕/屏幕UI` 字段涉及成片上屏内容，默认只上目标市场语言，不上中文双语屏幕字，除非用户明确要求双语版

## 镜头分级与继承规则

### 镜头分级

**关键镜头（必须填写全部 14 字段）：**
- 钩子落地镜头
- 爽点/反杀/揭露/身份曝光核心镜头
- 尾钩硬切镜头
- 高潮定格/情绪峰值镜头
- 首次展示角色/场景的建立镜头

**标准镜头（可使用简化版 7 字段）：**
- 普通对话/反应镜头
- 过渡空镜
- 连续动作中的跟随镜头（非关键动作节点）

### 标准镜头简化格式

标准镜头可使用以下简化版，无需填写全部 14 字段：

| 镜头号 | 时长 | 景别 | 画面内容 | 台词/音频 | AI画面提示 | 连续性锁定 |
|-------|------|------|---------|----------|-----------|-----------|
| 1-1-03 | 2s | 中景 | 男主转身，表情冷漠 | "你觉得呢。" | ↑继承 + 男主背向镜头，慢转身，正面特写 | ↑继承 |

### 连续性锁定继承规则（↑继承）

同场内相邻镜头，若以下字段与上一镜相同，可以写 `↑继承` 代替重复填写：
- 连续性锁定（服装/伤势/站位/时间段不变时）
- 负面约束（同场全局约束不变时）
- AI画面提示中的场景锚点和人物锚点（角色外观+场景环境未变化时）

**禁止继承的情况：**
- 场景切换后的第一个镜头
- 角色服装/伤势/道具状态发生变化后
- 时间段跳跃后
- 关键镜头（必须独立填写完整字段）

## 统一硬字段

每一集分镜稿都必须包含：

- 集信息
- 场次信息
- 镜头明细
- 本集分镜统计
- AI 视频生成附录

每个镜头都必须包含以下字段：

| 字段 | 作用 | 关键镜头判级 | 标准镜头判级 |
|------|------|-------------|-------------|
| 镜头号 | 唯一定位 | P0 缺失阻塞 | P0 缺失阻塞 |
| 来源场次 | 回溯源剧本 | P0 缺失阻塞 | P0 缺失阻塞 |
| 预计时长 | 控制单集总时长 | P0 缺失阻塞 | P0 缺失阻塞 |
| 景别 | 控制镜头任务 | P0 缺失阻塞 | P0 缺失阻塞 |
| 机位/角度 | 控制视角 | P0 缺失阻塞 | P1 可走"↑继承" |
| 运镜 | 控制动态表达 | P0 缺失阻塞 | P1 可走"↑继承" |
| 画面内容 | 说明谁在做什么 | P0 缺失阻塞 | P0 缺失阻塞 |
| 情绪目标 | 说明该镜头想让观众感到什么 | P0 缺失阻塞 | P1 可走"↑继承" |
| 台词/音频 | 对白、OS、VO、SFX、BGM 落点 | P0 缺失阻塞（若该镜有声） | P0 缺失阻塞（若该镜有声） |
| 字卡/字幕 | 关键文字强化和安全区位置 | P1 无字卡时可空 | P2 |
| AI 画面提示 | 直接给视频模型的视觉 prompt（含 artStyle 关键词） | P0 缺失阻塞 | P0 简化版可走 ↑继承 |
| AI 动作提示 | 直接给视频模型的 motion prompt | P0 缺失阻塞 | P1 静态镜头可空 |
| 连续性锁定 | 防止角色、场景、道具漂移 | P0 缺失阻塞 | P0 简化版可走 ↑继承 |
| 负面约束 | 防止模型跑偏 | P0 缺失阻塞 | P1 同场可走 ↑继承 |
| 时间线归属 | 标注本镜属于哪条时间线（现实 / 闪回-N / 梦境 / 平行时空 / 失忆前 / 失忆后 等），含闪回/双时间线项目必填 | 含闪回/双线项目 P0 缺失阻塞；纯单时间线项目可空 | 同上；同场连续可走 ↑继承 |

**完整度阻塞规则**：

- 关键镜头任意一项 P0 字段缺失 → `/分镜质检` 直接判 P0，禁止进入 `/导出`
- 标准镜头 P0 字段缺失（且无 ↑继承上一镜可承接） → 判 P0
- 关键镜头 P1 字段缺失 → 判 P1，强建议补齐
- 全集所有镜头逐项核对，必须形成"P0 字段全绿"才算交付完成

## 时长硬约束

> **权威定义在 `references/storyboard-conversion.md` §5"时长守恒规则"**，本节为摘要。

### 核心原则

镜头时长服从叙事需要，自下而上推导：镜头时长 → 场次时长 → 全集时长。不允许因为全集目标写了某个数字就强制压缩镜头时长。`本集目标时长` 应标注"按镜头叙事需要 · 实际 {N}s"。

### 闭合约束

每一集分镜稿必须同时满足以下约束，否则不算完成稿：

1. 每场 `本场预计时长` = 该场所有镜头 `时长` 求和，允许误差不超过 `0.5s`
2. `本集目标时长` = 全集所有场次时长之和（按叙事需要推导），允许误差不超过 `0.5s`
3. 若存在 `节拍备注` 或同类中间草稿数字，必须与最终镜头时长一致，不允许保留旧数字
4. 单镜头超过 `6s` 时，必须在 `画面内容` 或 `AI动作提示` 中交代该镜头为何值得停留（对白密度镜头可直接标注台词字数和语速）
5. 单镜头不得超过 `15s`（AI 视频模型单次生成硬上限），超出 = P0
6. 每个关键镜头都应能回答"这个时长依据是什么"；若只是为了补秒数，应删减或重拆
7. 对白镜头时长必须覆盖台词说完所需时间（按 `storyboard-conversion.md` 对白校准公式计算）；台词说不完 = P0

建议在每个场次信息下补一行：

- 时长校验：`镜头求和 {X}s / 场次目标 {Y}s / 状态：一致`

## AI 视频分镜模板

适用于本 skill 全部项目。无论 artStyle 是写实电影感、港风复古、国潮水墨、赛博朋克、厚涂插画还是 3D 卡通，分镜字段结构都相同；artStyle 只影响 `AI 画面提示` 的关键词。

> **文件头部必须包含 YAML frontmatter**（锁定源剧本版本，用于 `/总检` 断点对齐检查）。`source_episode_version` 必须与拆镜时 `分集剧本/第{NNN}集.md` 的 `version` 字段严格一致；若源剧本 version 被更新（比如 `/分镜脚本` 白名单回写或作者直接修订）而本分镜稿 `source_episode_version` 未同步，`/总检` 判 P0 "裸分镜集"，必须重走 `/分镜脚本 {起止集}` 或显式确认本分镜稿无需重做后手动升级 `source_episode_version`。

## 交付检查

交付前逐项确认：

1. 是否每个关键镜头都带 `AI画面提示` 和 `AI动作提示`，且 `AI画面提示` 已注入当前 artStyle 关键词
2. 是否每个镜头都带连续性锁定和负面约束
3. 是否关键字卡都写了安全区位置
4. 是否总镜头时长仍落在单集时长内
5. 是否尾钩镜头能单独生成并直接拿来做封口
6. 是否每场 `本场预计时长` 与场内镜头求和一致
7. 海外模式下，成片上屏字卡是否只保留目标市场语言，而中文仅留在文档中译
8. 是否每个关键镜头都能说清时长依据，而不是为了凑秒数停留
9. 是否所有镜头都服从本集全局节奏，而不是单场局部自洽
10. AI 视频生成附录里 `当前 artStyle` 和 `目标 AI 视频模型偏好` 是否与 `AI资产圣经.md` 全局生成规则一致
