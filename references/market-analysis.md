## 目标

立项阶段只回答五个问题：

1. 谁会点开？
2. 为什么会持续看？
3. 这类题材在 TikTok/海外短剧里靠什么赢？
4. 这项目为什么值得现在做？
5. 观众主要来买哪一种情绪？

## 推荐分析框架

### 1. 平台适配

平台适配的硬门槛和阻塞条件统一以 `references/tiktok-platform-constraints.md` 为准。

这里额外判断的是：这个方向在市场上“为什么值得推”，而不重复定义平台硬规则。

- 是否适合竖屏短时长？
- 是否能用高频 close-up 和情绪反应镜头成立？
- 是否有可重复出现的标志性动作或情境？

### 2. 受众动机

- 爽感：看主角翻盘、压制、夺回权力
- 情感：看心动、误解、修复、背叛
- 幻想：看身份跃迁、宿命绑定、超自然设定
- 紧张：看倒计时、秘密暴露、追查真相

立项时要明确主卖点属于哪一类，不要四类平均分。

### 3. 市场筛选

优先选择同时满足以下三项以上的项目：

- 标题即卖点
- 前 3 集能兑现小回报
- 中段能持续升级而不是重复吵架
- 角色关系一眼可懂
- 可设计强卡点
- 视觉或身份符号鲜明

## 候选方向打分卡

`/开始` 阶段不要只凭感觉推荐。至少对主推方向和备选方向做一次 10 分制快速评分：

| 维度 | 判断问题 | 分值 |
|------|---------|------|
| 点击力 | 标题或一句话 premise 是否 3 秒可懂 | 0-10 |
| 留存力 | 前 3 集是否能给到明确回报 | 0-10 |
| 升级力 | 中段是否有天然升级机制 | 0-10 |
| 关系炸点 | 角色关系是否一眼可懂且可持续拉扯 | 0-10 |
| 竖屏适配 | 是否适合 close-up、对峙、揭露、打断 | 0-10 |
| 低成本复用 | 核心场景与角色是否可高频复用 | 0-10 |
| 卡点转化 | 是否容易设计强卡点与付费节点 | 0-10 |
| 合规弹性 | 高风险点是否少，且可改写空间足够 | 0-10 |

推荐规则：

- 主推方向总分原则上不低于 60 分
- 任一方向若 `点击力`、`留存力`、`升级力` 中有两项低于 6 分，不应主推
- `合规弹性` 低于 5 分的方向默认不进入 `/立项`
- 若两个方向总分接近，优先选择 `竖屏适配 + 低成本复用` 更强的一项

## 应淘汰的立项

- 核心爽点只能靠解释成立
- 强依赖本地政策、官场制度、行业黑话
- 世界观复杂但情绪弱
- 主要冲突需要 10 集以后才真正开始
- 只有设定，没有人际关系炸点
- 题材带来明显平台或合规高风险，且不可替换

## `/开始` 阶段的方法论补充

> **执行规则归属**：`/开始` 命令的执行流程、启动入口、输出落盘规则、未确认配置规则、内建自检触发等以 `references/commands/start.md` 为唯一权威源。题材推荐池的具体内容、关键词补全、出海映射以 `references/start-topic-pool.md` 为唯一权威源。本节只补充 market-analysis.md 独有的"方向推荐方法论"，不重复执行规则。

每个候选方向至少要覆盖以下三层：

### 1. 建议配置 5 维度

每个候选方向除题材外，还要顺带给出建议配置：

- **建议版本**：出海版 / 国内版
- **建议艺术风格**：从 `references/ai-production.md` 的 artStyle 开放列表中给出 1-2 个候选（例如 写实电影感 / 港风复古 / 国潮水墨 / 赛博朋克 / 厚涂插画 / 3D 卡通），最终 artStyle 由 `/设定` 阶段锁定
- **建议市场**：北美 / 欧洲 / 东南亚 / 拉美 / 中东 / 混合
- **建议规模**：总集数 + 单集时长（默认 60-90 秒竖屏 AI 视频）
- **语言建议**：中文策划，英文对白版 / 全中文版本

本 skill 的最终交付一律是 60-90 秒竖屏 AI 视频，不区分"真人/漫剧/混合"管线；差异只来自艺术风格。

### 2. 推荐理由四问（必须每条都答）

- 为什么有人会点开？
- 为什么前 3 集能留住人？
- 为什么适合 TikTok 竖屏？
- 为什么能低成本反复拍？

四问任一回答含混 = 该方向不应作为主推。

### 3. 风险提示三类（至少覆盖一类）

- 文化适配风险
- 合规风险
- 制作成本风险

风险提示空缺 = 推荐方案不完整。
