# `故事设定.md` 固定模板

> 调用场景：`/设定` 首次创建。19 个子块按使用频率分 3 档，不要一口气全部 dump。

## 子块分档

**A. 必读子块（`/设定` 首次创建全部填写）**
§1.1 项目概览 / §1.2 世界规则 / §1.4 核心冲突 / §1.5 角色档案 / §1.6 角色卖点与出卡点 / §1.7 关系网络 / §1.9 角色命名与身份硬事实锁 / §1.11 场景池 / §1.12 角色外观锚点 / §1.13 全局视觉基调 / §1.14 场景视觉锚点 / §1.19 禁写项

**B. 题材条件子块**
- §1.3 能力边界 — 仅 Supernatural/Fantasy 题材；现实题材跳过
- §1.15 道具首登场清单 — 仅含关键道具的项目
- §1.16 角色外观变化记录 — 仅含伤势/换装/形象大变的项目
- §1.17 秘密与信息差 — 仅含多重秘密结构的项目

**C. 阶段条件子块**
- §1.8 角色情绪引擎 — 可延后到 `/结构` 或 `/分集大纲` 补填
- §1.10 角色台词声线卡 — 可在 `/设定` 初稿留空，但**最迟进入 `/分集剧本` 前必须补齐核心角色**；这是 `/剧本质检` clean-context 盲测和 `/分镜脚本` Voice Print 检查的唯一权威源
- §1.18 本地化方案 — 仅 `mode=overseas` 必填；国内模式跳过

**子块定位方法**：每个子块头严格按 `### §1.N 标题` 标注，用 Grep 锁定子块头再 Read 偏移；不要整个代码块 dump。
