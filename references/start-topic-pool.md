# 启动题材推荐池

## 目标

给 `/开始` 阶段提供一套稳定可复用的题材推荐池，避免每次都临场现编。重点解决两件事：

1. 用户没有明确想法时，能快速给出像样的候选方向。
2. 用户只给了模糊关键词时，能把想法补全成"题材组合 + 主情绪 + 受众 + 艺术风格"的可执行方向。

## 使用方式

- 本文件只服务 `/开始`
- 推荐时优先从下表中挑选，再结合 `references/market-analysis.md` 的评分卡做比较
- 每轮推荐优先给不同情绪、不同艺术风格的 3 个方向，避免 3 个候选只是同一题材轻微改名
- 本 skill 的最终交付一律是 60-90 秒竖屏 AI 视频；下表的 `建议艺术风格` 列只是常见参考。具体 `artStyle` 在 `/设定` 阶段先写入 `故事设定.md` 项目概览作为预锁字段，首次进入 `/分镜脚本` 创建 `AI资产圣经.md` 时继承并锁定全局生成规则

## 二、默认推荐顺序

### 1. 用户完全没想法时

按下面顺序挑主推：

1. 出海女频写实向：优先 `豪门反杀`、`秘密宝宝`、`黑帮暗恋`
2. 出海女频奇幻/半写实向：优先 `狼人伴侣`、`王室继承`
3. AI 原生强风格向：优先 `都市怪谈`、`异能学院`、`神明契约`、`漫画分格悬疑`
4. 国内女频现实向：优先 `都市复仇`、`家庭反击`、`霸总萌宝`
5. 国内男频爽点向：优先 `系统逆袭`、`修仙逆袭`、`末日重生`
6. 用户要求"想做点不一样的"：优先 `软科幻悬恋` 或 AI 原生强风格向

默认给 3 个候选时，除非用户明确只要真人感/现实题材，至少包含 1 个非写实或强风格 AI 原生方向，避免推荐池自动滑回单一写实实景路径。

### 2. 用户只给模糊关键词时

按“关键词 -> 补全方向”的方式处理：

| 用户关键词 | 补全方式 |
|-----------|---------|
| 霸总 / 豪门 | 补成 `Billionaire/CEO + 身份落差 + 反击/认子/契约关系` |
| 狼人 / 吸血鬼 / 奇幻 | 补成 `超自然身份 + 命定关系/被拒后逆袭 + 权力争夺` |
| 萌宝 / 带球跑 | 补成 `Secret Baby + 重逢旧爱 + 身份揭秘` |
| 复仇 / 打脸 | 补成 `公开羞辱 + 身份翻盘 + 阶段性清算` |
| 重生 / 穿越 | 补成 `已知未来 + 改命行动 + 连续碾压回报` |
| 系统 / 升级 | 补成 `面板机制 + 任务奖励 + 反派反应` |
| 悬疑 / 查真相 | 补成 `关键秘密 + 每集新线索 + 情感或身份代价` |

要求：

1. 不直接停留在单个题材词。
2. 必须补成"题材组合 + 主情绪 + 目标受众"的完整方向。

## 三、题材叠加规则

1. 默认只叠加 2-3 个元素：主题材 + 关系机制 + 升级机制。
2. 若世界观复杂，最多保留 2 个核心卖点，不要同时上奇幻体系、豪门权谋、悬疑探案三套系统。
3. 若主卖点是情感，不要再让复杂设定抢戏；若主卖点是升级，不要把大量篇幅耗在低效误会上。

推荐公式：

- 女频出海：身份题材 + 强关系机制 + 公开反击
- 男频升级向：升级机制 + 轻量世界观 + 连续打脸
- 国内女频：现实冲突 + 家庭/职场关系 + 快速兑现

## 五、/开始 交互动作

1. 用户完全没想法：
   先从推荐池里给 1 个主推和 2 个不同类型备选，候选之间要有明显情绪差异。
2. 用户有模糊想法：
   先复述理解，再把原始想法补全成 1 个主推修正版和 1-2 个变体。
3. 用户已经给出具体题材：
   不重复推荐大池，只校正是否适合 TikTok，并补足目标受众、情绪、形态。
4. 每轮只推进 1-2 个选择：
   优先确认题材方向，再给出建议艺术风格（最终 artStyle 由 `/设定` 阶段写入 `故事设定.md` 预锁，并在首次 `/分镜脚本` 创建 `AI资产圣经.md` 时继承锁定）。
5. 推荐时不要只说“这个题材火”：
   必须说明为什么点开、为什么留存、为什么可复用、为什么能持续升级。
