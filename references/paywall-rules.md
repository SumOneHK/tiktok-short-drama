# 付费卡点设计规则

## 总原则

1. 卡点前必须先给价值，再要转化。
2. 卡点不是简单“断在最高潮”，而是断在“下一步结果最不可逆”的位置。
3. 卡点必须建立在状态已变化，而不是结果还没开始变化。
4. 同一部剧不要只使用一种卡点手法。
5. 不能用低俗、未成年人风险、自残威胁等高风险内容做卡点。
6. 强卡点最好同时具备三件事：情绪峰值、后果阈值、结果未显。

## 位置建议

按总集数粗分：

- 30-40 集：首个强卡点建议在第 6-8 集
- 45-60 集：首个强卡点建议在第 8-12 集
- 60-80 集：首个强卡点建议在第 10-15 集

前提：

- 在首个强卡点前，观众已经看见至少一次明确兑现
- 首个卡点后 1-3 集内必须给到更大回报

## 卡点类型

### 1. 结果前置型

观众已经知道行动发生，最想看结果。

适合：

- revenge
- billionaire
- 悬疑追查
- power fantasy

示例场景：

- 身份即将公开
- 交易即将反杀
- 婚礼即将被打断

### 2. 关系翻页型

观众最想看两人关系是否越界。

适合：

- romance
- fake marriage
- forbidden love
- secret baby

示例场景：

- 即将接吻但被打断
- 误会即将揭开
- 一方听见半句真相

### 3. 代价落锤型

观众知道主角做了危险决定，想看代价是否爆发。

适合：

- mafia
- supernatural
- thriller
- chosen one

示例场景：

- 变身被目击
- 违规使用力量
- 保护某人导致身份暴露

### 4. 选择分叉型

观众要看角色站哪边，以及会失去什么。

适合：

- family melodrama
- emotional drama
- betrayal

示例场景：

- 在爱人与家族之间选边
- 在孩子、秘密、地位之间做选择

## 题材差异

### 复仇 / 逆袭 / 权力升级

- 卡点优先：结果前置型、代价落锤型
- 最强转化点：公开打脸前一秒、身份揭露前一秒、反杀局启动后但未落地
- 避免：一再延迟反击，导致观众疲劳

### Romance / Fake Marriage / Secret Baby

- 卡点优先：关系翻页型、选择分叉型
- 最强转化点：误会将破未破、欲望已越界但关系未命名、孩子身份即将被确认
- 避免：只靠亲密镜头，不给关系实质变化

### Werewolf / Vampire / Supernatural

- 卡点优先：代价落锤型、结果前置型
- 最强转化点：觉醒、bond 暴露、等级反转、诅咒启动
- 避免：规则解释过长，稀释情绪

### Suspense / Investigation

- 卡点优先：结果前置型、信息钩子型
- 最强转化点：新证据出现、真凶切换、看似无辜者露馅
- 避免：假线索过多，导致观众不再相信线索

## 反模式

- 结果还没启动就硬断
- 每次都断在“有人来电话”
- 卡点后迟迟不给兑现
- 把真正的剧情推进都堆到付费后
- 用低俗刺激替代结构性卡点
- 纯动作打斗打到一半就断，但情绪和结果都没被建立
- 只断在“快亲上了”，却没有关系实质变化

## 总检问题

在 `/总检` 阶段至少检查：

1. 首个强卡点前是否已有阶段性回报？
2. 每个 `💰` 是否对应明确的转化理由？
3. 卡点类型是否与题材匹配？
4. 卡点后是否在 1-3 集内给出更强回报？
5. 是否存在拖延兑现或重复同类卡点？
